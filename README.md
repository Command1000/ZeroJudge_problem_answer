# ZeroJudge Solutions

This repository is used to store my source code for practice problems on the **ZeroJudge** online judge platform. It serves as a personal record of my problem-solving progress and algorithmic practice.

## Contents
* **Source**: Solutions to basic and competitive programming problems from ZeroJudge.
* **Language**: Primarily written in C++.
* **Naming Convention**: Files are named after the problem ID (e.g., `a001.cpp`, `k731.cpp`).

## Tools Used
* **Editor**: Vim
* **Compiler**: g++
* **Version Control**: Git
